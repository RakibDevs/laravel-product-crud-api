# Laravel CRUD API with JWT Auth
Laravel product CEUD API with JWT auth using repository pattern.

## Initial Setup
```
git clone https://github.com/rakibdevs/laravel-product-crud-api.git
cd laravel-product-api
cp .env.example .env
composer update
php artisan migrate
php artisan serve
```
